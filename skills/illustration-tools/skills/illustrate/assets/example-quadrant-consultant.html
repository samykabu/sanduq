THE HEADLINE

Name every future

Scenario planning fails when three of the four cells are placeholders. Forcing a name on each future — even the "won't happen" ones — surfaces the assumptions hiding in the axes.

Variant rules

  • Double-ended arrows on both axes (spectrum, not flow)
  • Named cells, not plotted items
  • Driver name + range on each arrow tip
  • Coral on one focal quadrant max

When to use

Reach for the consultant 2×2 for scenario planning, positioning frames, or strategy sessions where four named archetypes beat a plotted point cloud. If you're prioritising, use the standard quadrant instead.
